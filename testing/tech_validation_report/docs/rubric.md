# 评分量表（1-5）

## 1. 主题相关性
5：始终紧扣主题，无离题内容，对主题理解精准
4：基本扣题，偶有轻微泛化但不影响整体
3：大部分相关，但存在部分偏离或泛化段落
2：多处跑题或大量泛泛而谈
1：严重偏题，与给定主题关联微弱

## 2. 大纲遵循度
5：严格遵循大纲，所有章节/页面/要点一一对应
4：基本遵循大纲，个别页面顺序或要点有微小偏差
3：大体遵循，但部分页面内容与大纲要点不匹配
2：偏离明显，章节缺失或增删较多
1：与大纲结构严重不符

## 3. 内容完整性
5：覆盖所有关键要点，信息密度高，有深度分析
4：覆盖主要要点，少量细节可补充
3：覆盖基本要点，但存在明显信息缺口
2：内容单薄，关键要点缺失较多
1：内容严重不足，无法支撑汇报需求

## 4. 结构清晰度
5：章节逻辑清晰，页面间过渡流畅，单页层次分明
4：结构基本清晰，个别过渡可优化
3：整体可读，但存在逻辑跳跃或层次混乱
2：结构松散，逻辑线不清晰
1：结构混乱，难以跟随

## 5. 知识补充利用程度
5：知识引用精准恰当，与内容深度融合，无堆砌感
4：知识使用合理，引用基本恰当
3：使用了知识但部分引用生硬或未充分利用
2：知识使用不足或盲目堆砌，与内容脱节
1：未利用提供的知识/文档，或引用严重错误

## 6. 事实一致性风险（5=风险最低）
5：所有关键陈述均有证据支撑，无自相矛盾
4：绝大多数陈述有支撑，偶有无证据断言但风险低
3：部分陈述缺乏支撑，但无明显错误
2：多处无证据断言或存在轻微矛盾
1：存在明确事实错误或严重自相矛盾

## 7. PPT 展示可用性
5：几乎可直接展示，少量润色即可
4：需要少量调整（措辞或顺序）
3：需要中度修改（重排或补内容）
2：需要大量修改才能可用
1：需要大幅重写才可用于 PPT
