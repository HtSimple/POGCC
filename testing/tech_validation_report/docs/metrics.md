# 指标定义

## 1. 主题相关性（1-5）
- 内容是否紧扣给定主题，不跑题、不泛化
- 是否准确理解主题的核心诉求

## 2. 大纲遵循度（1-5）
- 生成内容是否严格遵守大纲结构
- 章节顺序、页面数量、关键要点是否对应

## 3. 内容完整性（1-5）
- 是否覆盖了主题与受众所需的关键要点
- 信息密度与深度是否充足

## 4. 结构清晰度（1-5）
- 章节划分是否逻辑清晰
- 页面间叙事是否流畅、无跳跃
- 单页内层次是否分明

## 5. 知识补充利用程度（1-5）
- 是否有效利用检索到的知识/上传文档补充内容
- 证据引用是否恰当、不盲目堆砌

## 6. 事实一致性风险（1-5，5=风险最低）
- 是否存在与材料矛盾或自相矛盾的陈述
- 无证据支撑的断言数量与严重程度

## 7. PPT 展示可用性（1-5）
- 少量修改是否即可直接用于 PPT 展示
- 语言风格、篇幅、排版适配度
